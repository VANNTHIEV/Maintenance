# អត្ថប្រយោជន៍​របស់ **Windows Automatic Maintenance**

## 1. បង្កើន​ System Performance

Windows នឹងជួយសម្អាត **temporary files**, បង្កើនល្បឿន **disk
optimization**, ហើយធ្វើឱ្យកុំព្យូទ័ររបស់អ្នកដំណើរការលឿនជាងមុន។

## 2. ការពារ PC ឱ្យមានសុវត្ថិភាព

Automatic Maintenance នឹងដំណើរការ\
- **security updates**\
- **antivirus scan**\
- ការកែសម្រួល **security vulnerabilities**

ធ្វើឱ្យប្រព័ន្ធរបស់អ្នកមានសុវត្ថិភាពជានិច្ច។

## 3. ជួសជុលបញ្ហាដោយស្វ័យប្រវត្តិ

Windows នឹងពិនិត្យ និងជួសជុល\
- **system errors**\
- **corrupted files**\
- **crashes**\
- បញ្ហា **driver issues**

ដោយអ្នកមិនចាំបាច់ធ្វើអ្វីទេ។

## 4. Updates សម្រាប់ Apps និង Windows ដោយស្វ័យប្រវត្តិ

Apps និង Windows components នឹងត្រូវបាន update ដោយឆ្លាតវៃ\
ដោយមិនចាំបាច់ update ដោយដៃ។

## 5. កាត់បន្ថយការងារដោយដៃ

អ្នកមិនចាំបាច់ធ្វើជា\
- **disk defragment**\
- **junk cleanup**\
- **system diagnostics**\
- ពិនិត្យ **updates**

Windows ធ្វើចប់អស់ទាំងអស់។

## 6. ដំណើរការតែនៅពេល PC Idle

Automatic Maintenance មិនរំខានអ្នកទេ។\
វាដំណើរការនៅពេល\
- PC នៅក្នុងស្ថានភាព **idle**\
ហើយវានឹងផ្អាកភ្លាមៗនៅពេលអ្នកប្រើប្រាស់វិញ។
